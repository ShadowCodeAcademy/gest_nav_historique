# Système de navigation historique (gest_nav_historique)

## Contexte
La navigation web utilise des structures linéaires pour gérer les boutons Précédent / Suivant.

## Objectifs pédagogiques
- Listes doublement chaînées
- Gestion dynamique de mémoire
- Alternatives par piles

## Fonctionnalités
- Visiter une page
- Aller à la page précédente
- Aller à la page suivante
- Supprimer l’historique
- Recherche d’URL
- Limitation de taille
